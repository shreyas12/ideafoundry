## Rules
- **≤3K.** This is a summary, not the plan. Feature detail lives in `01-product-requirements.md`; tickets in `12-developer-tickets.md`. Link, don't inline.
- **Aggregate risks** across every present specialist's risk field — never copy one file's list.

- Absent-file gating: no `decisions` file → drop *Open decisions*; no `goals` file → fall back to `brief.goals`. Never emit a broken `product.*`/`decisions.*` reference.
